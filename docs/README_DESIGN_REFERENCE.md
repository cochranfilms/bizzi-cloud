# Design Reference for Graphic Designers

## Creating a PDF

Open `DESIGN_REFERENCE.html` in a browser (Chrome, Safari, Edge), then:
1. **File → Print** (or Ctrl/Cmd+P)
2. Choose **Save as PDF** as the destination
3. Click Save

## Files

- `DESIGN_REFERENCE.md` — Source markdown
- `DESIGN_REFERENCE.html` — Print-ready HTML with color swatches (open in browser to save as PDF)
